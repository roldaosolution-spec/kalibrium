# Autonomia de Ambiente — agentes

**O usuário `paperclip` tem `sudo NOPASSWD`.** Agentes podem instalar qualquer coisa via apt/npm/docker/etc.

Documentem instalações em `ENVIRONMENT.md`.

Se algo NÃO puder ser resolvido via sudo (API keys externas, domínio, decisão comercial), abrir issue `ops:` ou `board:` no Paperclip.
