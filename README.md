# Kalibrium

> SaaS multi-tenant para laboratórios de calibração e empresas de serviço técnico em campo — construído autonomamente por um time de agentes IA orquestrados via Paperclip.

## Status
**Em reconstrução do zero.** O PRD está em [`docs/PRD.md`](docs/PRD.md).

## Stack (ADR-0001)
- PHP 8.4 + Laravel 13
- PostgreSQL 18
- PWA + Capacitor (mobile offline-first)
- Livewire 4 + Alpine.js + Tailwind CSS 4

## Processo
Código produzido por agentes autônomos do Paperclip e revisado por um **audit panel de 10 especialistas** (isolados, sem contexto cruzado) antes de virar PR. Revisões humanas são raras e pontuais.
